## Full Description (4000 chars max)
Privacy Shield is a comprehensive security and privacy tool for Android that helps you understand and protect your digital environment.

**Device Detection**
Scan for nearby WiFi and Bluetooth LE devices in real time. Privacy Shield identifies 13 device types including cameras, microphones, routers, IoT devices, phones, and more. Each device gets a risk classification and distance estimate.

**Privacy Score**
Get an instant privacy score (0-100) based on the types and proximity of detected devices. The score updates in real time as new devices are discovered.

**Network Analysis**
Connect to any WiFi network and analyze it thoroughly:
• Host discovery — find all devices on your subnet
• DNS leak check — verify your DNS isn't leaking
• Security analysis — open WiFi detection, gateway exposure, IP privacy
• Evil twin detection — identify rogue access points trying to impersonate legitimate networks

**Security Tools**
A full suite of network security tools:
• Port Scanner — scan any host for open ports with risk labels
• Ping — test connectivity with visual response graphs
• WHOIS Lookup — get IP geolocation and ISP information
• Traceroute — trace the path to any host
• CVE Lookup — search the NVD vulnerability database for any vendor or product
• Network Scanner — discover live hosts and services on your network

**Privacy & Security**
• Biometric app lock — protect the app with fingerprint or face unlock
• Background scanning — monitor your environment automatically
• Push notifications — get alerted when suspicious devices are detected
• Scan history — review and export previous scan sessions

**No ads. No tracking. No cloud uploads.**
All scanning happens locally on your device. No data is sent to external servers except for optional CVE and WHOIS lookups.
